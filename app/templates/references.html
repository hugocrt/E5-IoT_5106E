This is a project carried out as part of my studies at ESIEE Paris, in the course IoT-5106E taught by Nawel ZANGAR ATTIA.

CSV satellites dataset found on CelesTrak.

I used this python library to do the calculations for the satellites Python Skyfield Library.

Apps and Tools

PyCharm

PyCharm was used as the primary development environment for writing and debugging Python code.

GitHub

GitHub was used for version control, enabling tracking of changes.

Docker

Docker allowed for containerization of the application, ensuring consistent deployment in different environments.

Programming languages

Frontend Development

HTML5, CSS, and JavaScript were used to create the web interface, ensuring users could interact with the data easily.

Backend Development

- Python was the main programming language for backend logic, handling satellite data processing and integration with Elasticsearch.
- Flask was used as the micro-framework to develop the web application, facilitating communication between the frontend and backend.
- Elasticsearch was used as the search and analytics engine, enabling efficient indexing, querying, and retrieval of satellite pass data. It allowed for fast data storage and retrieval, making it easy to search and filter satellite information based on various parameters.
